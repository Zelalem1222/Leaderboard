# LEADERBOARD

## Built With

- HTML
- CSS 
- Javascript
- webpack
- Gitflow


## Getting Started

To get a local copy up and running follow these simple example steps.

- Clone repository in your local machine 
- cd Portfolio
- open `index.html` in your browser.
- You can use live server.
